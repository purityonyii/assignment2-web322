## Project Description
This project is Assignment #2 of WEB322.  
It includes:
- Server-side form validation (registration & login)
- MVC structure (models, views, controllers)
- express-session for authentication
- bcrypt for password hashing
- MongoDB Atlas database connection
- Welcome email capability (MailGun optional)
- Protected routes for customer & data clerk roles
- Responsive styling using Bootstrap
